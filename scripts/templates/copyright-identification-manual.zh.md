## 二、软件概述

本软件为面向 **WPS 文字** 的 **智能加载项**，在 Ribbon 功能区与右键菜单中集成人工智能辅助能力，使用户可在文档编辑环境中完成对话、文本分析、翻译、多模态内容生成、保密与脱敏相关辅助、文档与表格及图像的批量处理、表单辅助与规则管理等操作；生成结果支持以插入、替换、批注等方式写回当前文档。

软件基于现代 Web 前端技术实现界面与业务逻辑，通过 WPS 提供的 JSAPI 与宿主文档交互；模型侧支持 **OpenAI 兼容协议** 的 HTTP 接口，便于对接本机或内网部署的推理服务（如 Ollama、LM Studio、Xinference、OneAPI 等）以及主流云端大模型服务。

## 三、运行环境与依赖

1. **操作系统**：与 WPS 文字所支持的桌面操作系统一致（常见为 Windows；若使用 macOS/Linux 版本 WPS，以实际安装环境为准）。
2. **宿主软件**：已正确安装的 **WPS 文字** 客户端。
3. **网络**：按需访问用户自行配置的模型 API 地址；纯离线场景可仅使用内网或本机 OpenAI 兼容端点。
4. **开发与打包（维护人员）**：Node.js（建议 LTS）、npm；可选 wpsjs 与 WPS JSAPI 工具链用于加载项调试与打包。

## 四、安装与加载

1. 取得与本版本对应的加载项安装包或部署目录（由出品方发布的在线/离线包为准）。
2. 在 WPS 中按「开发工具 / 加载项」或企业 IT 规定的途径完成加载项安装或信任路径配置。
3. 启动 WPS 文字并打开文档后，在功能区应出现本软件相关选项卡或按钮组；若未出现，请检查加载项是否被禁用或是否需以管理员策略放行。

## 五、主界面与入口

- **功能区（Ribbon）**：提供关于、AI 助手入口、任务编排、任务清单、文本与表格及图像批量处理、模板与规则、设置等分组入口。
- **右键菜单**：在选中内容上可快速调用「添加到察元」、文本分析、翻译、智能助手等能力。
- **对话框与任务窗格**：部分能力以非模态或模态对话框、任务窗格形式呈现，便于在编辑文档的同时查看进度与结果。

## 六、AI 助手与对话

1. 通过功能区或入口打开 **察元 AI 助手** 对话窗口。
2. 在设置中配置 **API 基础地址**、**API 密钥**（若供应商要求）及 **模型名称**；离线类供应商通常可不填密钥。
3. 在对话区输入问题或指令；支持流式输出时，界面将逐字显示模型回复。
4. 可将回复内容 **插入、替换** 至当前选区或文档指定位置，或以 **批注** 形式附加说明。

## 七、模型与供应商设置

1. 打开 **设置**，进入模型或供应商相关页面。
2. 为每个启用的供应商填写 **API URL**（通常以 `/v1` 结尾或按供应商文档填写）与 **API Key**。
3. 在模型列表中勾选需要在 Ribbon 或助手中展示的模型；未启用的模型将不在选择列表中出现。
4. 支持 **providerId|modelId** 形式的组合标识，以便同一网关下区分多个模型。

## 八、文本分析、翻译与智能助手

- **文本分析**：提供改写、扩写、缩写、校对、关键词提炼、段落序号检查、AI 痕迹检查、批注与超链接解释等能力（具体以界面为准）。
- **翻译**：支持多语言方向；语言列表可由配置或内置菜单生成。
- **智能助手**：支持固定入口与「更多」动态菜单；可启动预置助手任务并在任务清单中查看历史与结果。

## 九、文档写回与安全相关

1. **写回方式**：插入、替换、追加至文末、仅批注、批注并替换等；具体可用项取决于当前上下文与助手配置。
2. **保密与脱密**：提供保密检查、文档脱密、脱密复原等入口（以实际界面为准）；涉及敏感文档时请在合规前提下使用。
3. **任务与进度**：长时间任务将创建任务记录，可在任务清单或进度对话框中查看状态、取消或重试。

## 十、表格、图像与文档批量

- **表格**：支持导出/删除全部表格、按内容或窗口调整列宽、按文字删除行列、追加替换、首列序号、题注等批量操作。
- **图像**：支持导出/删除全部图像、统一或清除格式、题注等。
- **文档**：支持清理未使用样式、样式使用统计、删除空白行等。

## 十一、表单、模板与规则

1. **模板**：支持从文档导出/导入规则、下载模板等流程。
2. **规则与书签**：表单项与文档书签关联，用于表单辅助填报与编审场景。
3. **预览与审计**：可对表单内容进行预览，并支持文档审计类流程（以界面说明为准）。

## 十二、开发与构建（维护说明）

维护人员可在仓库根目录执行 `npm install` 安装依赖；使用 `npm run dev` 启动开发服务；使用 `npm run build:wps` 等命令进行加载项打包。具体命令以 `package.json` 中 `scripts` 字段为准。

## 十三、常见问题

1. **无法连接模型**：检查 API 地址、密钥、网络及防火墙；内网环境需确保终端可访问网关地址。
2. **功能区无按钮**：检查加载项是否启用、WPS 版本是否支持、是否需修复安装。
3. **写回失败**：确认当前文档未处于只读或受限保护状态，且选区有效。
